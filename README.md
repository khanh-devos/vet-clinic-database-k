# 📖 Vet-Clinic <a name="about-project"></a>

Vet-Clinic is a PostgreSQL database of a vet clinic, which includes Names, Dates of birth, Escape attempts, Neutering status, and Weight.

## 💻 Getting Started <a name="getting-started"></a>

This repository includes files with plain SQL that can be used to recreate a database:

- Use [schema.sql](./schema.sql) to create all tables.
- Use [data.sql](./data.sql) to populate tables with sample data.
- Check [queries.sql](./queries.sql) for examples of queries that can be run on a newly created database. **Important note: this file might include queries that make changes in the database (e.g., remove records). Use them responsibly!**

To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need:

`install postgresql.`

### Setup

Clone this repository to your desired folder:

`git clone https://github.com/khanh-devos/vet-clinic-database-k.git`

### Install

Install this project with:

```sh
  cd vet-clinic-database-k
```

### Usage

To run the project, execute the following command:

```sh
  rails server
```

### Run tests

To run tests, run the following command:

`sh bin/rails test test/models/article_test.rb`
